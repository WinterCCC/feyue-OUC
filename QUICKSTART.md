# 飞跃手册快速开始指南

## 🎯 项目一句话介绍
中国海洋大学飞跃手册是学生分享升学、出国、就业经验的知识库。

## 📂 核心文件速览

| 文件 | 用途 |
|------|------|
| `index.md` | 网站首页 - 最新文章和导航 |
| `_config.yml` | Jekyll 配置 - 站点名称和主题 |
| `about.md` | 项目介绍 - 使命和如何使用 |
| `CONTRIBUTING.md` | 投稿指南 - 详细的贡献步骤 |
| `CLAUDE.md` | AI 助手指南 - 项目架构说明 |
| `_posts/` | 文章目录 - 所有发布的文章 |

## 📝 发表新文章

### 3步快速发表

```bash
# 1. 创建文件（格式：YYYY-MM-DD-标题.md）
_posts/2024-12-20-my-article.md

# 2. 添加前置信息
---
title: "文章标题"
date: 2024-12-20
categories: [保研/考研/出国/就业]
author: 你的名字
---

# 3. 开始写内容
## 前言
...
```

```markdown
---
title: "完整标题"

categories: [分类]
author: 作者名
---

```

## ✍️ 写作建议

### ✅ 好的内容
- 真实具体的经历
- 包含时间和数据
- 提供可操作的建议
- 分享失败的教训
- 推荐具体资源

### ❌ 避免

- 夸大或虚假信息
- 纯理论总结
- 没有实际例子
- 过于冗长

## 📊 内容示例对比

### ❌ 不够好
> 我通过保研成功进入了一所好学校。准备很重要。

### ✅ 很好

## 🚀 从想法到发表

### Week 1: 筹备
- 确定主题和分类
- 收集经验和资料
- 整理时间线

### Week 2: 撰写
- 按模板架构文章
- 填充各部分内容
- 加入具体数据和建议

### Week 3: 修改
- 检查拼写和格式
- 请朋友审阅
- 完善表述

### Week 4: 发表
1. Fork 仓库
2. 创建分支并添加文件
3. 提交 Pull Request
4. 等待审核（3-7 天）
5. 发表！

## 📋 投稿检查清单

在提交 PR 之前，确保：

- [ ] 文件名格式：`YYYY-MM-DD-title.md`
- [ ] Front matter 完整：title, date, categories, author
- [ ] 至少有一个合适的分类
- [ ] 内容真实可信
- [ ] 没有拼写错误
- [ ] 包含具体的时间和数据
- [ ] 提供实用建议
- [ ] 包含作者简介
- [ ] 已本地预览检查格式

## 💡 常见问题

**Q: 我没有成功的经历，能投稿吗？**
A: 完全可以！失败的经历和反思同样有价值。

**Q: 多久能看到发表？**
A: 审核通常 3-7 天，审核通过后立即发表。

**Q: 可以修改已发表的文章吗？**
A: 可以，提交新的 PR 提议修改即可。

**Q: 需要写得很专业吗？**
A: 不需要。真诚、清晰就是最好的。
