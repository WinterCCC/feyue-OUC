# 欢迎来到海大飞跃手册！

这是由中国海洋大学学生会创建的**经验分享平台**，汇集了海大学子在升学、出国、就业等方面的真实故事和宝贵经验。

## 📚 手册特色

- **真实经历分享** - 学长学姐的第一手经验
- **多元升学路径** - 保研、考研、出国、就业全覆盖
- **实用建议指导** - 避坑指南和成功秘诀
- **社区互动交流** - 为学弟学妹答疑解惑

## 💬 如何投稿？

欢迎各位海大校友投稿分享你的升学、出国或就业经历！

**投稿步骤：**
1. Fork 本仓库
2. 在 `_posts` 目录下创建文件，格式为 `YYYY-MM-DD-title.md`
3. 按照模板编写文章（见下方示例）
4. 提交 Pull Request

**文章模板：**
```markdown
---
title: "你的标题"
date: 2024-12-01
categories: [保研/考研/出国/就业]
author: 你的名字
---

文章内容...
```
