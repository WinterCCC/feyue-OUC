## 📚 项目简介

**中国海洋大学飞跃手册** 是由海大学生发起的**非营利性知识共享平台**，旨在为全校学生提供升学、出国、就业等方面的真实经验和实用建议。

### ✨ 核心优势


| 特点           | 说明                             |
| -------------- | -------------------------------- |
| 🎓**真实可信** | 海大学子第一手经验，避免纸上谈兵 |
| 🌍**全面覆盖** | 保研、考研、出国、就业等全方位   |
| 📈**持续更新** | 每届新经验，及时反映政策变化     |
| 🤝**互动友好** | 学长学姐热心解答，建立学长制度   |

## 💝 如何贡献

### 📝 投稿文章

我们热烈欢迎你的投稿！

**你适合投稿如果**：

- ✅ 有保研、考研、出国或就业的成功经历
- ✅ 愿意分享真实的经验和教训
- ✅ 想为学弟学妹提供帮助

**投稿步骤**：

1. Fork 本仓库
2. 在 `_posts/` 创建文件（格式：`YYYY-MM-DD-title.md`）
3. 按照 [投稿指南](./CONTRIBUTING.md) 编写
4. 提交 Pull Request

📖 [详细投稿指南 →](./CONTRIBUTING.md)

## ❓ 常见问题

### Q: 我的经历失败了，还能投稿吗？

A: 完全可以。失败的经历和反思同样宝贵！

### Q: 投稿有稿费吗？

A: 没有，但会获得署名。这是一项公益项目。

### Q: 文章可以匿名投稿吗？

A: 可以，但建议署名以便读者交流。

### Q: 政策每年都在变，这些内容还有用吗？

A: 有的。虽然政策会变，但方法论和经验是通用的。我们也会持续更新最新政策。
