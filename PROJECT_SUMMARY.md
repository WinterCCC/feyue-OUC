## 🚀 快速使用指南

### 方式1：预览网站
```bash
# 直接打开预览文件
open preview.html
# 或在浏览器中打开 E:\文件\feiyue-ouc\feyue-OUC-main\preview.html
```

### 方式2：本地运行 Jekyll
```bash
# 安装依赖（首次）
bundle install

# 启动本地服务
bundle exec jekyll serve

# 访问 http://localhost:4000
```

### 方式3：发表到 GitHub Pages
1. 上传到 GitHub 仓库
2. 在 Settings → Pages 中启用 GitHub Pages
3. 访问 https://ouc-feiyue.github.io/

## 📝 投稿流程

### 投稿5步走
```
1. Fork 仓库
   ↓
2. 创建分支 (feature/your-topic)
   ↓
3. 创建文件 (_posts/YYYY-MM-DD-title.md)
   ↓
4. 按模板编写文章
   ↓
5. 提交 PR 等待审核
```

### 文章必须包含
- ✅ 正确的 Front Matter（title, date, categories, author）
- ✅ 个人背景介绍
- ✅ 具体的时间规划
- ✅ 可操作的建议和方法
- ✅ 常见问题解答
- ✅ 作者简介
